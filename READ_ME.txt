Read ME file for Sports4U

#Steps to Run the application: 
1. Copy the "Sports4U" folder and paste it into the "webapps" folder of tomcat directory 
2. Start the MySql server and create Database named - "exampledata" 
3. Then start the MongoDb Server and create Database "CustomerReviews" 
4. After both the database servers are started, then start the Tomcat server 
5. Run the application by writing localhost/Sports4U/Home in the browser
